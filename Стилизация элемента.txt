# Стилизация элемента 
Общая схема стилизации: 
Стилизация элемента.png

Для стилизации универсального селектора используем звёздочку ==*==. Это означает, что мы хотим стилизовать вообще все элементы в нашем документе. 
Универсальный селектор.png

Селектор элемента используется, чтобы выбрать какой-то конкретный элемент. Можно выбирать как один конкретный элемент, так и несколько через запятую. 
Селектор элемента.png

Выбор по id. Указываем решётку ==#== и добавляем название индитификатора. ID отвечает за уникальную маркировку элемента. На странице может быть только один. 
Выбор по id.png

Если id предназначен для стилизации одного конкретного элемента, то выбор по классу позволяет работать сразу с группой элементов. То есть мы объединяем эти элементы в единый класс и затем разом применяем ко всем им единый стиль. Синтаксис: ставим точку, указываем название класса и в скобках пишем стили. 
Класс элемента.png

Выбор по атрибуту помогает сужать выбор элемента, который мы хотим стилизовать. Например: мы хотим стилизовать тег `input` с атрибутом `type='text'`:
