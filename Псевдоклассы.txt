# Псевдоклассы 
Псевдоклассы позволяют выбрать элемент в определённом состоянии или позиции. Например у нас есть элемент `button` и мы хотим чтобы при наведении менялся цвет текста и самой кнопки. Синтаксис: ==название элемента: {новый стиль}==. 
Пример работы псевдокласса.png

Псевдоклассы с параметрами. Позволяют более точно указать, какой элемент мы хотим выбрать для стилизации. Так можно стилизовать каждый чётный элемент: 

```
ul li:nth-child(even){
    color: orange;
  }
```

А так каждый нечётный: 
```
ul li:nth-child(odd){
    color: orange;
  }
```

А вот мы выбираем каждый третий элемент: 
```
ul li:nth-child(3n){
    color: orange;
  }
```

Подобные записи удобно использовать когда мы хотим покрасить строки таблицы или выполнить какой-то другой чередующийся дизайн. 

Параметры внутри скобок можно представлять в виде формул, начинать примерять стили как с начала, так и с конца. Допускается любая последовательность.
